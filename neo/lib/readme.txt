The following libraries are included in the Neo Framework distribution because they are
required either for building the framework or for running the Visual Studio Plug-In. Note 
that each of these libraries is subject to the respective license; check the respective 
project distribution/website before using any of them in your own applications.

* BaseCodeGeneratorWithSite.dll
- http://www.gotdotnet.com/Community/UserSamples/Details.aspx?SampleGuid=4AA14341-24D5-45AB-AB18-B72351D0371C
- required for using the VisualStudio Plug-In

* Commons.dll
- Jakarta Commons 0.4.2, .NET port from NVelocity (http://nvelocity.sourceforge.net/)
- required for using the VisualStudio Plug-In

* ConnectionFactory.dll
- A .NET connection factory (http://docs.codehaus.org/display/NEO/FooConnectionFactory)
- required for building the framework and most datastores
- required at runtime

* FirebirdSql.Data.Firebird.dll
- Firebird 1.6.1 client libraries (http://firebird.sourceforge.net/)
- required at runtime when using the Firebird datastore

* ICSharpCode.SharpZipLib.dll 
- SharpZipLib 0.81 (http://www.icsharpcode.net/OpenSource/SharpZipLib/)
- required at runtime when using the MySQL datastore

* log4net.dll
- log4net 1.2 (http://logging.apache.org/log4net/)
- required for building the framework
- required at runtime

* microsoft.visualstudio.designer.inerfaces.dll
- required for using the VisualStudio Plug-In

* Mono.GetOptions.dll
- Mono GetOptions 1.0 (http://www.mono-project.com/)
- required for using the command-line generator

* Mono.Security.dll
- Mono Security 1.0 (http://www.mono-project.com/)
- required at runtime when using the PostgreSQL datastore

* MySql.Data.dll
- MySQL client libraries 1.0.4 (http://www.mysql.com/)
- required at runtime when using the MySQL datastore

* nmock.dll
- NMock 1.0 (http://www.nmock.org/)
- required for building and running the framework test suite

* Npgsql.dll
- PostgreSQL client libraries 0.7 (http://www.postgresql.org/)
- required at runtime when using the PostgreSQL datastore

* nunit.core.dll
- NUnit 2.2 (http://www.nunit.org/)
- required for building and running the framework test suite

* nunit.framework.dll
- NUnit 2.2 (http://www.nunit.org/)
- required for building and running the framework test suite

* NVelocity.dll
* NVelocity 0.4.2 (http://nvelocity.sourceforge.net/)
- required for using the VisualStudio Plug-In
