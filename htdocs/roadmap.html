What we are working on

Documentation

This includes a more comprehensive Quick Start guide and API documentation. Simon Williams made a good start at the latter and it looks like we'll be using Ndoc to embed the documentation into the actual code.

A new SQL generator

With the refactoring of the ER model classes complete we will create a SQL generator based on Velocity templates. The XSL based generator will continue to work as it only depends on the XML format of the model file; which will not be changed.

Model Diagrams

Paul is adding a new generator to the tool set that will generate output that can be transformed into an Entity Relationship Diagram using Dot.

Planned Features

Improved qualifiers

Qualifiers that can contain set restrictions. Have a look at the following examples which should be self explanatory:

new TitleAuthorfactory().Find("Title[Royalty > 5000].Publisher.Country = 'Something'");

Maybe other OPath features as well.

Schema generation for 'legacy' databases

Sometimes you will have the need to retrofit an object layer using Neo to an existing database. In such a case it would be cool to generate either the XML file or the object model from schema information in the database. A first step would be to add Neo support to MiddleGen.
