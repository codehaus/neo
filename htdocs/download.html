Distributions

Download the latest distributions from

http://dist.codehaus.org/neo/distributions

Note that we currently only have source code distributions which require VisualStudio 2003 to build. If anybody creates NAnt script please send a note to the neo-users mailinglist.

Anonymous CVS access

This project's CVS repository can be checked out through anonymous (pserver) CVS with the following commands. When prompted for a password for anonymous, simply press the Enter key.

cvs -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/neo login

cvs -z3 -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/neo co neo

Updates from within the module's directory do not need the -d parameter.

Note that the source distributions already contain the required CVS directories. All you need to do to keep up to date after downloading a distribution is to log in using the first command above (once) and then use the following command from within the toplevel neo directory:

cvs -z3 update neo

Browse source

You can browse the source code and all its versions online at:

http://cvs.codehaus.org/viewcvs.cgi/?root=neo

This might help you decide whether you want to update to a new version.
