Distributions

Download the latest distributions from

http://dist.codehaus.org/neo/distributions

Note that we currently only have source code distributions which require VisualStudio 2003 to build. If anybody creates NAnt script please send a note to the neo-users mailinglist.

Anonymous CVS access

This project's CVS repository can be checked out through anonymous (pserver) CVS with the following commands. When prompted for a password for anonymous, simply press the Enter key.

cvs -d :pserver:anonymous@cvs.neo.codehaus.org:/scm/neo login

cvs -z3 -d :pserver:anonymous@cvs.neo.codehaus.org:/scm/neo co neo

Updates from within the module's directory do not need the -d parameter.

The source distributions already contain the required CVS directories. All you need to do to keep up to date after downloading a distribution is to log in using the first command above (once) and then use the following command from within the toplevel neo directory:

cvs -z3 update neo

Important note: The repository has changed between the 1.1.1 and 1.2.0 releases. If you are working with a 1.1 release please download a zipped distribution or check out the new version as described above.
