Wiki Wiki Web

Neo space on docs.codehaus.org

Random rants and other assorted comments. The Wiki is writable for everybody and you are welcome to add comments; comments related to Neo that is.

Mailing Lists

user@neo.codehaus.org

Discussions about the use of Neo and its tools.

dev@neo.codehaus.org

Discussions pertaining to the future development of Neo itself; not about the development of systems using Neo.

scm@neo.codehaus.org

Automated messages generated by CVS. This includes all changes to the Neo codebase.

Bugs/Issues

Browse

Browse bugs and other issues on the Codehaus issue tracking system. Note that you won't be given the option to log new bugs and issues unless you have logged into the system.

Report

Once you have created an account and logged in to the Codehaus issue tracking system on the main page you can report new issues here. Please make sure that you select 'Neo' from the drop down menu.
