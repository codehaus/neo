Presentations

Neo_DesignGoals.ppt (160kb)

Describes the design goals for Neo, mostly in the form of code snippets that outline how Neo would be used in an application.

Neo_Architecture.ppt (112kb)

An overview of Neo from a design perspective. Explains how the design goals are achieved in the implementation. This covers all classes but does not explain how the Neo framework is used. This is left as an exercise to the reader...

Neo_BestPractices.ppt (50kb)

Early draft of a presentation outlining best practices when developing applications with Neo.
