Is this all?

No, most of the FAQ lives on the Neo Wiki FAQ pages. Note that you are welcome to add new questions to the wiki pages.

How do I report bugs?

Please use the Jira Issue Tracker to report bugs. This makes it much easier to refer to them from other places. You will have to create an account to report bugs but that only takes a few moments and it allows us to contact you when we have questions regarding the bug you reported.

Where are they binaries?

At the moment there are no pre-build binaries. Neo is a developer tool and it is quite easy to build so we didn't consider an installer msi for binaries top priority. There should be one eventually, though.
